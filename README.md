### MIA vector search!

just install `requirements.txt` and run `cli.py`

to vectorise a work pass the root of the index (e.g. if you want to vectorise Capital, pass `https://www.marxists.org/archive/marx/works/1867-c1/`, NOT `https://www.marxists.org/archive/marx/works/1867-c1/index.htm` nor `https://www.marxists.org/archive/marx/works/1867-c1`)

takes a while to vectorise long texts, but the search kinda works so thats neat
